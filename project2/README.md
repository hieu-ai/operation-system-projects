To compile:

In xv6-kernel-threads

Step 1: Install qemu:

$ sudo apt-get install qemu

Step 2: Compile xv6

$ make

Step 3: Compile and run the emulator qemu:

$ make qemu

or

$ make qemu-nox

Step 4: Run tests

$ threadtest1
$ threadtest2
$ threadtest3
$ mutextest1
$ mutextest2
